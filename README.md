# WatchWarden App Store Web Deployment

This folder is generated by `scripts/package_appstore_web.sh`.

Deploy the folder contents to the configured GitHub Pages host.

## Required Public URLs

- `https://conorgg123.github.io/watchwarden-app-site/support/`
- `https://conorgg123.github.io/watchwarden-app-site/privacy/`
- `https://conorgg123.github.io/watchwarden-app-site/privacy-choices/`
- `https://conorgg123.github.io/watchwarden-app-site/terms/`

The package includes clean route folders for hosts that serve `index.html`,
plus `_redirects` for Netlify-style routing, `vercel.json` rewrites for Vercel,
and `.nojekyll` for GitHub Pages.

After deployment, verify with:

```sh
scripts/check_external_release_gates.sh --report-only --check-urls
```
